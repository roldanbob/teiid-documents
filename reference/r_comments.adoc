[id="comments"]
= Comments

{{ book.productnameFull }} supports multi-line comments enclosed with `/* */`.

[source,sql]
----
/* comment
comment
comment... */
----

You can also add single line comments:

[source,sql]
----
SELECT ... -- comment
----

You can also nest comments.
